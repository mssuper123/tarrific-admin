# tarrific-admin

tarrific.tv DSP 管理后台（单页 Vue 应用）。

## 本地预览

直接用浏览器打开 `index.html`，或使用本地静态服务：

```bash
python3 -m http.server 8080
# 访问 http://localhost:8080
```
